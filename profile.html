Tóm tắt

Hơn 10 năm kinh nghiệm trong kế toán – tài chính, trực tiếp phát triển các giải pháp kế toán doanh nghiệp tại Việt Nam (sản phẩm chính: VACOM). Trong 3 năm gần đây, mở rộng sang phát triển ứng dụng mobile bằng React Native, chuyên về quản lý tài chính, CRM và nghiệp vụ doanh nghiệp. Kinh nghiệm publish app lên Google Play & App Store và triển khai app nội bộ cho doanh nghiệp.

Dự án tiêu biểu

  • Ứng dụng quản lý hộ kinh doanh tại Việt Nam – React Native (Đã public)
    Quản lý doanh thu, chi phí, báo cáo tài chính.
    Link tải: Google Play / App Store
  • Ứng dụng CRM nội bộ công ty – React Native (Nội bộ)
    Quản lý khách hàng, giao dịch, lịch hẹn.
    Đóng góp: frontend + logic API + triển khai cho nhân viên nội bộ.
  • Ứng dụng quản lý kế toán trên mobile – React Native (Đang thực hiện)
    Hỗ trợ quản lý nghiệp vụ kế toán doanh nghiệp trực tiếp trên điện thoại.

Kỹ năng chính

  • React Native, Expo, TypeScript
  • State management: Redux Toolkit, React Query
  • API integration, push notification, offline sync
  • CI/CD, deploy lên App Store & Google Play
  • Tối ưu performance, debug, UX cơ bản

Mục tiêu

Hỗ trợ các task phát triển ứng dụng mobile, từ fix bug, thêm feature đến full app, đặc biệt liên quan đến kế toán, CRM, quản lý doanh nghiệp.
